# Select

드롭다운 단일 선택. reka-ui `SelectRoot` 위에 디자인 시스템 스타일을 입힌 compound 컴포넌트.

## 구성

| 서브 컴포넌트 | 역할 |
|---|---|
| `<Select>` | Root. v-model 바인딩 |
| `<SelectTrigger>` | 현재 값 + chevron. 버튼 역할 |
| `<SelectValue>` | 현재 선택된 value 텍스트 (`placeholder` 지원) |
| `<SelectContent>` | 드롭다운 팝오버 |
| `<SelectItem>` | 각 옵션 |
| `<SelectGroup>`, `<SelectLabel>`, `<SelectSeparator>` | 그룹 헤더/구분 |

## 기본 사용

```vue
<Select v-model="selected">
  <SelectTrigger>
    <SelectValue placeholder="선택하세요" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="kr">대한민국</SelectItem>
    <SelectItem value="us">미국</SelectItem>
    <SelectItem value="jp">일본</SelectItem>
  </SelectContent>
</Select>
```

## FieldContainer와 함께 (권장)

```vue
<FieldContainer name="country" label="국가" required>
  <Select>
    <SelectTrigger>
      <SelectValue placeholder="선택하세요" />
    </SelectTrigger>
    <SelectContent>
      <SelectItem value="kr">대한민국</SelectItem>
      <SelectItem value="us">미국</SelectItem>
    </SelectContent>
  </Select>
</FieldContainer>
```

FieldContainer 컨텍스트가 있으면 `v-model`을 생략해도 자동 바인딩. 없으면 `v-model`로 연결.

## 트리거 스타일

기본 `h-[40px] px-[16px] text-size-14`, default variant와 동일 색상 체계 (grey-10 bg, grey-40 border, focus blue-80).

현재는 **regular 단일 사이즈**. small/large 필요 시 FieldContainer의 size 전파가 아닌 커스텀 class로 해결.

## Do / Don't

✅ **Do** — 필수 조합 사용
```vue
<SelectTrigger>
  <SelectValue placeholder="..." />
</SelectTrigger>
```

❌ **Don't** — SelectTrigger에 직접 텍스트 박기
```vue
<!-- SelectValue 없으면 placeholder 미동작 -->
<SelectTrigger>선택</SelectTrigger>
```

## 접근성

reka-ui 기반이라 WAI-ARIA combobox 패턴 자동 적용:
- `aria-haspopup="listbox"`, `aria-expanded` 관리
- 키보드: 위/아래 화살표, Enter, Space, Escape, Home/End
- 타입 투 셀렉트 (typeahead) 지원
