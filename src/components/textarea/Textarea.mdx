# Textarea

여러 줄 텍스트 입력. Input과 동일한 variant/size 체계에, **auto-resize**와 **counter 내부 배치**가 추가됩니다.

## Variant & Size

- Variant: `default` / `filled` / `bottomline`
- Size: `small` / `regular` / `large`

Figma 스펙 regular: `p-16 text-14`. large: `p-16 text-16`.

## 주요 props

| Prop | 기본 | 설명 |
|---|---|---|
| `autoResize` | `true` | 내용에 따라 높이 자동 조정 |
| `minHeight` / `maxHeight` | `60 / 300` | auto-resize 범위 |
| `rows` | `3` | autoResize가 false일 때 고정 줄수 |
| `counter` + `maxLength` | - | **container 내부 하단 우측**에 카운터 렌더 (Figma textCounter variant) |
| `byteMode` | `false` | counter 계산을 byte 기준으로 |

## 상태

Input과 동일 (disabled 시 텍스트 grey-50, readonly 시 grey-80).

## Do / Don't

✅ **Do**
```vue
<FieldContainer name="memo" label="메모" description="최대 500자">
  <Textarea placeholder="내용을 입력하세요" counter :max-length="500" />
</FieldContainer>
```

❌ **Don't**
```vue
<!-- deprecated props 사용 금지 -->
<Textarea label="메모" error-message="..." />
```

## 접근성

- `inheritAttrs: false` + attrs forwarding으로 FieldContainer의 aria 연결 보장
- resize는 `resize-none` 기본 (auto-resize가 담당). 사용자 리사이즈 핸들은 의도적으로 비활성
