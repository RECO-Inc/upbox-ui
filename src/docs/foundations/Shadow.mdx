# Shadow

4단계 elevation 토큰으로 공간 깊이를 표현합니다.

| 토큰 | CSS 변수 | 용도 |
|---|---|---|
| `shadow-small` | `--shadow-small` | 카드, 얕은 elevation (리스트 아이템 hover 등) |
| `shadow-regular` | `--shadow-regular` | 일반 카드, 버튼 눌림 feedback |
| `shadow-medium` | `--shadow-medium` | 팝오버, 툴팁 |
| `shadow-large` | `--shadow-large` | 모달, 드롭다운, FAB |

## 사용

```html
<div class="bg-grey-10 rounded-[8px] shadow-regular p-[16px]">
  카드 내용
</div>
```

## 원칙

- **Overlay일수록 큰 shadow** — Modal(large) > Popover(medium) > Card(regular)
- **focus shadow는 별도 토큰 아님** — Tailwind `focus-visible:ring-*` 활용 (Input blue-40 ring 등)
- **Dark mode에서는 자연스럽게 약화됨** — shadow 토큰이 light/dark 분기 가짐
