# Upbox UI

Vue 3 컴포넌트 라이브러리. Figma 디자인 시스템을 코드로 구현한 내부 디자인 시스템.

## 무엇이 들어있나

- **컴포넌트**: Button, Field, Picker, Dialog/Drawer/Sheet/Popover, Table, Sidebar 등 50+개
- **디자인 토큰**: Figma 팔레트(grey/navy/blue/green/red/orange) → CSS 변수 → Tailwind 유틸리티
- **합성 패턴**: InputFrame 기반 Field, Compound Picker, Monolithic + Compound dual API
- **모바일 변형**: 동일 입력 컴포넌트를 데스크톱 Popover / 모바일 Drawer 로 분기

모두 Tailwind v4 + `@tailwindcss/vite` 사용. 셋업 디테일은 Installation 가이드 참고.

## 배포 방식

`dist/` 가 git 에 포함되어 있고 컨슈머는 **GitHub URL 직접 참조** 방식으로 설치:

```bash
pnpm add git+https://github.com/RECO-Inc/upbox-ui.git
```

별도 npm registry 발행 없음. `main` 브랜치에 push → 컨슈머에서 `pnpm remove` 후 재설치로 업데이트.
