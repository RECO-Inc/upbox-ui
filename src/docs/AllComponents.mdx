# All Components

Upbox UI 디자인 시스템의 **모든 컴포넌트를 한 페이지에서** 확인하는 오버뷰 문서입니다.
각 항목은 대표 예시 하나만 보여줍니다 — 전체 variant·prop·상태는 좌측 `Components/` 카테고리의 개별 스토리에서 확인하세요.

카테고리 구성:

| # | 카테고리 | 포함 |
|---|---------|------|
| 1 | **Buttons & Actions** | Button, TextButton, IconButton, Fab, Toggle, Toolbar |
| 2 | **Forms & Inputs** | TextField, Textarea, NumberField, SearchField, Checkbox, RadioGroup, Form, FileUploader |
| 3 | **Date & Time** | DatePicker, DatePeriodPicker, TimePicker, DateCalendar, DateMove, Mobile\* |
| 4 | **Data Display** | Avatar, Badge, BadgeDivider, Tag, FilterChip, Card, Table, DataTable, Panel, Empty, Skeleton, Spinner, Logo, Separator |
| 5 | **Navigation** | Breadcrumb, Tabs, Pagination, Stepper, PageHeader, Accordion, Sidebar |
| 6 | **Overlays & Feedback** | Dialog, Modal, Drawer, Sheet, Popover, Tooltip, DropdownMenu, DropdownFilter, Toast |
