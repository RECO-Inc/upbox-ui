# Installation

## 설치

GitHub URL 직접 참조 방식 (npm registry 미발행):

```bash
# pnpm (권장)
pnpm add git+https://github.com/RECO-Inc/upbox-ui.git

# npm
npm install git+https://github.com/RECO-Inc/upbox-ui.git
```

> 업데이트 시 pnpm 캐시 때문에 그냥 `pnpm update` 로는 최신 main 이 안 가져와집니다. `pnpm remove @reco-inc/upbox-ui && pnpm add git+...` 로 다시 받기.

## Tailwind v4 셋업

Tailwind v4 는 **CSS-first 설정** 방식입니다. 컨슈머의 CSS 엔트리에서 다음 4줄만 추가하면 됩니다.

```css
/* style.css */
@import "tailwindcss";
@import "@reco-inc/upbox-ui/style.css";          /* 컴파일된 라이브러리 CSS */
@import "@reco-inc/upbox-ui/theme.css";          /* @theme 토큰 등록 */
@source "../node_modules/@reco-inc/upbox-ui/dist/**/*.mjs";
```

각 줄이 하는 일:

| 라인 | 역할 |
|------|------|
| `@import "tailwindcss"` | Tailwind v4 코어 |
| `@import ".../style.css"` | 라이브러리의 모든 컴포넌트가 사용하는 컴파일된 CSS (reset 포함) |
| `@import ".../theme.css"` | 디자인 토큰을 `@theme` 블록에 노출 → `bg-grey-80` 등의 유틸리티 활성화 |
| `@source ".../dist/**/*.mjs"` | dist JS 안의 클래스 문자열까지 스캔해야 미사용 클래스가 purge 되지 않음 |

(`@source` 경로는 CSS 파일 위치 기준 상대경로)

### Vue 템플릿 자동완성 (권장)

`env.d.ts` 또는 `vite-env.d.ts` 에 추가:

```ts
/// <reference types="@reco-inc/upbox-ui/global" />
```

→ `.vue` 의 `<template>` 안에서 `<Button>` 같은 라이브러리 컴포넌트가 import 없이도 인텔리센스에 잡힙니다.

## 사용 예시

```vue
<script setup lang="ts">
import { Button, Card, CardHeader, CardTitle, CardContent } from '@reco-inc/upbox-ui'
</script>

<template>
  <Card>
    <CardHeader>
      <CardTitle>Title</CardTitle>
    </CardHeader>
    <CardContent>
      <Button variant="primary" size="regular">Click</Button>
    </CardContent>
  </Card>
</template>
```

## 다크 모드

`<html class="dark">` 또는 `<body class="dark">` 클래스 토글만으로 모든 색 토큰이 자동 반전됩니다 (CSS 변수 레이어에서 처리).

## 자주 묻는 문제

### 컴포넌트 색이 안 입혀짐

→ `@reco-inc/upbox-ui/theme.css` 가 import 안 됐거나, `@source` 에 dist 경로가 빠짐. 위 4줄을 모두 확인.

### 업데이트해도 옛 버전이 잡힘

→ pnpm 의 Git URL 캐싱. `pnpm remove @reco-inc/upbox-ui && pnpm add git+...` 로 강제 재설치.

### 컴포넌트 수정 후 컨슈머에 반영이 안 됨

라이브러리 측에서:

```bash
npm run build          # dist/ 재생성
git add dist/ src/     # dist 도 같이 커밋
git commit -m "..."
git push
```

컨슈머 측에서:

```bash
pnpm remove @reco-inc/upbox-ui
pnpm add git+https://github.com/RECO-Inc/upbox-ui.git
```
